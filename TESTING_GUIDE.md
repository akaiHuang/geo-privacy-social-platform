# 快速測試指南

## 🚀 本地測試步驟

### 1. 啟動開發伺服器
```bash
cd web
npm run dev
```

### 2. 測試現有功能（確保地圖未被破壞）
- 訪問 `http://localhost:5173/`
- 測試動態流
- **重點測試地圖功能**：
  - 切換到地圖檢視
  - 測試地圖互動
  - 測試時光回溯功能
  - 確認所有地圖功能正常

### 3. 測試新的個人檔案功能

#### 測試 URL 路由
訪問以下 URL（替換為實際的 username）：
```
http://localhost:5173/@[username]
```

#### 測試未登入體驗
1. 在無痕視窗打開個人檔案頁面
2. 瀏覽貼文，注意每 3 篇會顯示登入提示
3. 點擊貼文應觸發登入提示
4. 測試「繼續瀏覽」按鈕
5. 測試「立即登入」按鈕

#### 測試已登入體驗（需要進一步開發）
- 當前已登入用戶將看到基本檔案
- 完整互動功能需要進一步整合

### 4. 測試錯誤處理
訪問不存在的用戶：
```
http://localhost:5173/@nonexistentuser123
```
應該顯示「用戶不存在」訊息

## 🔍 檢查清單

- [ ] 首頁載入正常
- [ ] 地圖功能完全正常（**最重要**）
- [ ] 時光回溯功能正常
- [ ] 可以訪問 `/@username` 路由
- [ ] 個人檔案頁面載入正常
- [ ] 未登入時顯示登入提示
- [ ] 不存在的用戶顯示錯誤訊息
